# Smart Clock v0.1.0

**Release Date:** May 30, 2026

## Highlights
- Complete Roman numeral clock with candle-flicker night mode
- DS3231 RTC + AHT20 environmental sensor
- US SAME code weather radar + Amber/Silver alert system with photo display
- Smart scheduler (3-hour weather updates, 5-min during alerts)
- Voice announcements every 15 minutes during active alerts
- Full-featured Web UI (mobile-friendly, dark mode, config editor)
- MQTT with Home Assistant auto-discovery, subscriptions, and LWT
- OTA updates via Web and MQTT (with safe rollback + USB fallback)
- Rotary encoder time setting and menu navigation
- SD card support (persistent alarms, sounds, backups)
- Energy monitoring for Alexa smart outlets
- Multi-language support (English, Spanish, German, French)

## Hardware
Raspberry Pi Pico 1 + Pico DVI Sock + ESP8266 WiFi
