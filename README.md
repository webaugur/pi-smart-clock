# π Smart Clock

A beautiful fullscreen Rust + SDL2 clock for Raspberry Pi (framebuffer) with:
- Large Roman numeral analog clock with mechanical second hand + tick/tock sounds
- Bottom panel: Calendar | Holidays | Weather

## Features
- Runs directly on framebuffer (no X11)
- Modular bottom panels
- Easy to extend

## Setup
```bash
cargo run
```

Built with ❤️ for the Pi 3 / Pi 4.
